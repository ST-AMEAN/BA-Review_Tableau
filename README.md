# British Airways Customer Experience Analytics Dashboard


## Overview
Developed an interactive Tableau dashboard analysing British Airways passenger reviews from 2016–2023. The dashboard provides a dynamic exploration of trends in customer satisfaction, segmented by demographics and geographic specifics. It features key performance metrics such as service quality, overall satisfaction, and value for money. Interactive elements allow users to filter data based on year, region, and type of flight, offering a customizable analysis experience.

## Business Problem
Airlines rely heavily on customer feedback to evaluate service quality and identify improvement opportunities. However, large volumes of review data can make it difficult to understand key drivers of customer satisfaction.
This project aims to transform passenger review data into an interactive business intelligence dashboard that supports data-driven decision-making and customer experience analysis.


## Objectives
- Analyse customer satisfaction trends over time.
- Compare service quality metrics across different categories.
- Identify regional differences in passenger ratings.
- Evaluate aircraft performance based on customer feedback.
- Enable interactive exploration through dynamic filters.


## Dashboard Features
### Customer Satisfaction KPIs
Track key service metrics including:
- Overall Rating
- Cabin Staff Service
- Entertainment
- Ground Service
- Food & Beverages
- Seat Comfort
- Value for Money

### Geographic Analysis
- Explore average customer ratings by country.
- Identify regional differences in passenger experience.

### Time-Series Trend Analysis
- Monitor customer satisfaction trends from 2016–2023.
- Detect periods of improvement or decline.

### Aircraft Performance Comparison
- Compare customer ratings across aircraft models.
- Evaluate aircraft popularity through review volume
  
### Interactive Filtering
Users can analyse results by:
- Rating Category
- Date Range
- Traveller Type
- Seat Type
- Aircraft
- Continent


## Key Insights
- Customer satisfaction varied considerably across countries and regions.
- Cabin Staff Service consistently received stronger ratings than Entertainment and Value for Money.
- Value for Money and Entertainment are among the lowest-rated service dimensions.
- Aircraft performance differs across fleet types, highlighting opportunities for service improvement.
- Customer satisfaction trends fluctuate over time, providing insights into changing passenger expectations.
- Service quality dimensions contribute differently to overall customer satisfaction.
